# NEXUS | Futuristic GitHub Explorer

Nexus is a premium, high-performance GitHub API integration platform built with modern frontend technologies. It features a stunning glassmorphism UI, real-time data fetching, and an intelligence-dashboard aesthetic designed for recruiters and tech enthusiasts.

## 🚀 Key Features
- **Real-Time Intelligence:** Fetches live user profiles and repository data from the GitHub REST API.
- **Glassmorphism UI:** Advanced CSS techniques for transparency, blur, and neon glow effects.
- **Responsive Architecture:** Fully optimized for Mobile, Tablet, and Desktop using CSS Grid and Bootstrap 5.
- **Dynamic Theming:** Smooth transitions between Futuristic Dark and Sleek Light modes.
- **Interactive UX:** Skeleton loaders, scroll progress bars, and high-fidelity modals.

## 🛠️ Tech Stack
- **HTML5:** Semantic structure for SEO and accessibility.
- **CSS3:** Custom properties (variables), Flexbox, Grid, and Keyframe animations.
- **Vanilla JavaScript (ES6+):** Object-Oriented architecture for state management and API handling.
- **Bootstrap 5:** Grid system and modal utilities.
- **Font Awesome:** Premium iconography.

## 📂 Project Structure
```text
├── index.html      # Main entry point & UI structure
├── style.css       # Design system & futuristic styling
├── script.js       # Core application logic & API service
└── assets/         # Resource storage
```

## 🎯 Implementation Details
- **API Service:** Uses `fetch` with `async/await` for asynchronous data handling.
- **State Management:** Simple internal state tracking for current user and repository data.
- **Performance:** Minimized dependencies and optimized animations for 60fps rendering.

## 🎨 Design Inspiration
Inspired by modern SaaS dashboards and cyberpunk aesthetics, focusing on high-contrast neon accents against deep space-blue backgrounds.
